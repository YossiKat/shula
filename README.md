1. הכנס לתיקיית `/bots`.
2. צור קובץ חדש, למשל `real_estate_expert.txt`.
3. כתוב בפנים את ההנחיות (Prompts).
4. בצע Push ל-GitHub. שולה תזהה את הבוט החדש אוטומטית בדשבורד.
